# AI-Powered Business Case Study Generator

A 2-week portfolio project to convert raw company information into structured, source-backed business case studies.

## Setup

1. Create virtual environment
2. Install requirements
3. Run Streamlit app

```bash
pip install -r requirements.txt
streamlit run app/streamlit_app.py

## Companies and Sources

We currently support three demo companies:

- Infosys – integrated annual report + HR case study.
- Nykaa – investor presentation, annual return, founder stories.
- Zerodha – open-source report, narrative deck, news article, origin story, and official page.

See `data/source_metadata.csv` for full source list and types.
```
